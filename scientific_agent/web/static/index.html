How the evidence gate works

Qwen plans, retrieves, computes, and drafts. The controller locks the method, confines Python/R and package installation, records artifacts, and runs deterministic checks. Gemma then independently audits methods, claims, tables, and actual raster figures. Gemma is the only model that receives images; Qwen never performs visual interpretation. Tests and source records outrank model agreement.

Biomedical and health-science analyses must search PubMed and acquire locally previewable article Markdown, with a verified PDF when legitimately available. Context7, Brave Search, and Chrome DevTools are enabled by default and Qwen is instructed to use the connection relevant to the task; users can uncheck any connection before starting. Every substantive claim remains linked to a retrieved source, successful computation, or explicit uncertainty. Cancellation preserves partial work as incomplete; a follow-up creates a new audited child revision and never overwrites its parent.

Outputs are standards-derived exploratory scientific reports—not peer review, science lock, manuscript readiness, or clinical authority.
